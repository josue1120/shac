S.H.A.C Instrucciones

Siguiendo unos simples pasos podrás saber cómo funciona esta plataforma y crear rápidamente una actividad.
1. Crea un nuevo usuario para que administre el panel.
2. Crea una sección para tus grupos.
3. Crea tus grupos o grados.
4. Crea las materias que ven tus grupos.
5. Crea una actividad.
6. Mira el calendario del grupo.
